# Net-tools

É um pacote que provê os comandos:

arp

ifconfig

ipmaddr

iptunnel

route

nameif

mii-tool

## Instalação

Toda a instalação é feita como **root**

### Debian

Instale o pacote net-tools com o comando:

 ` apt-get install net-tools`

### CentOS

Instale o pacote net-tools com o comando:

 ` yum -y install net-tools`
